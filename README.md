# 📚 Operating System Study
> UNIX is basically a simple operating system, but you have to be a genius to understand the simplicity (Dennis Ritchie)

## ⭐️ 진행방식
- 주 1회 오프라인 모임
- 스터디장 발표 후에 자율적으로 개인 발표시간 (TMI 대환영)
- 자료는 깃허브에 업로드 예정
- 단방향보다는 양방향을 지향 (질문 대환영)
- **함께 고민하고, 공부하는 시간**
